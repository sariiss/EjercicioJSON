Se necesita una manera de administrar la información del usuario, como las canciones disponibles y estado de reproducción ( el volumen, me gusta, estado actual de la aplicación ) de una forma organizada y facil de leer por la aplicación. Para esto se realizo un archivo JSON en el cual representa toda esta información en una estructura, permitiendo que la aplicación pueda acceder rápidamente a los datos del usuario, controlando y mostrando las canciones que está reprocuciendo sin que la información no esté desordenada.
